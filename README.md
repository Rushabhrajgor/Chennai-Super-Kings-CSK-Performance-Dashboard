# Chennai-Super-Kings-CSK-Performance-Dashboard
CSK Dashboard (Tableau)

Project Title:
CSK Performance Analytics Dashboard

Short Description:
This project presents an interactive Tableau dashboard analyzing the performance of Chennai Super Kings (CSK) across multiple seasons. It transforms raw cricket data into meaningful insights, helping users explore team performance, player statistics, and match trends through dynamic visualizations. Analyze CSK’s overall performance across seasons, Identify top-performing players in batting and bowling, Discover trends in wins, losses, and match outcomes, Provide an interactive platform for cricket data exploration.

Tools & Technologies:
Tableau – Data visualization & dashboard development,
Microsoft Excel / CSV – Data storage and preprocessing,
Data Cleaning & Transformation – Ensuring data accuracy,
Data Analytics – Extracting insights from datasets.

Key Features
Team performance summary (Wins/Losses),
Player statistics (Runs, Strike Rate, Wickets),
Season-wise performance trends,
Boundary analysis (Fours & Sixes),
Top performers identification,
Match insights and comparisons,
Interactive filters and drill-down capabilities.
